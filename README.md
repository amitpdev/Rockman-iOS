# RockMan

RockMan is a small iOS music player app and a tribute to my favorite game series of all time: Rockman / Mega Man.

I love the soundtracks from these games so much that I wanted a simple way to keep them available on the go, as an iOS app on my phone.

The app focuses on the classic 8-bit NES era of Rockman / Mega Man, where the music is fast, melodic, and instantly recognizable despite the hardware limits of the original console.

## Music

The track list is built around the music I most want to hear from the games:

- Robot Master boss stage themes, ordered by the original stage select screens.
- Dr. Wily castle themes and other late-game fortress tracks.
- Stage select music, title screen music, intros, endings, and credits.
- Game-specific artwork for boss stages, title screens, stage select screens, and ending scenes.

Audio is stored as high-quality original FLAC files where available, so the app keeps the soundtrack as clean as possible while still feeling like the original NES music.

## Running

This app is only for fun. It can be installed on an iPhone from Xcode by opening the project and running it on a connected device.

It will not be published to the App Store.
