# RockMan

RockMan is a small iOS music player app and a tribute to my favorite game series of all time: Rockman / Mega Man.

I love the soundtracks from these games so much that I wanted a simple way to keep them available on the go, as an iOS app on my phone.

## Running

This app is only for fun. It can be installed on an iPhone from Xcode by opening the project and running it on a connected device.

It will not be published to the App Store.
